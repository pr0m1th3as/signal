Function Reference: gauspuls

Function File: y = gauspuls (t)
Function File: y = gauspuls (t, fc)
Function File: y = gauspuls (t, fc, bw)

Generate a Gaussian modulated sinusoidal pulse sampled at times t. See also: pulstran, rectpuls, tripuls

Source Code: gauspuls

Example: 1

  

 fs = 11025;  # arbitrary sample rate
 f0 = 100;    # pulse train sample rate
 x = pulstran (0:1/fs:4/f0, 0:1/f0:4/f0, "gauspuls");
 plot ([0:length(x)-1]*1000/fs, x);
 xlabel ("Time (ms)");
 ylabel ("Amplitude");
 title ("Gaussian pulse train at 10 ms intervals");
